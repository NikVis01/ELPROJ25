PROJECT FOLDER FOR ELECTRO PROJECT 2025 

SETUP:

1. Open Arduino IDE

2. Go to File > Preferences

3. Set "Sketchbook location" to your project's parent folder

4. Enable "Use external editor" if using another code editor

5. Go to Tools > Board > Boards Manager

6. Search for your board (e.g., "Arduino uno") & click install

7. Select board: Tools > Board > [Your Board] (e.g., "Arduino Uno")
