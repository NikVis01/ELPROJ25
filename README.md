PROJECT FOLDER FOR ELECTRO PROJECT 2025 

FOR SETUP WITH ARDUINO IDE:

1. Configure Board
    Press Ctrl+Shift+P
    Run "Arduino: Board Config"
    Select your board (e.g., "Arduino Uno")
    Set port (e.g., /dev/ttyACM0)

2. Build and Upload
    Build: Ctrl+Alt+R
    Upload: Ctrl+Alt+U
    Monitor: Ctrl+Alt+Shift+M
